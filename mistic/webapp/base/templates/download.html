All files are in tab-separated value (TSV) format and contain 6 fields:

  1. CHROM : chromosome
  2. POS : position
  3. REF : reference nucleotide
  4. ALT : alternate nucleotide
  5. {{ config.FLASK_APP_NAME }}_score : {{ config.FLASK_APP_NAME }} score
  6. {{ config.FLASK_APP_NAME }}_hspred : {{ config.FLASK_APP_NAME }} "high sensitivity" prediction for exome analysis
  7. {{ config.FLASK_APP_NAME }}_pred : {{ config.FLASK_APP_NAME }} prediction for global performance
